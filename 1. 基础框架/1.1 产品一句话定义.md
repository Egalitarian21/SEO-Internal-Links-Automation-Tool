# Phase 0-1：产品一句话定义（PRD Seed）

## 任务描述
在开始任何 AI 辅助开发之前，用一句话精准锁定产品的核心价值主张。
这是整个 Vibe Coding 工作流的"北极星"，所有后续 AI Prompt 都应围绕此定义展开。
明确目标用户、核心功能、解决的痛点以及与竞品的差异化。

---

## Prompt

为 [目标用户] 构建一个 [核心功能] 的工具，解决 [核心痛点]，
区别于 [竞品] 的是 [差异化]。

请根据以上模板，帮我完善产品定义，并输出：
1. 一句话产品定位（≤30字）
2. 核心用户故事（3条 As a... I want... So that...）
3. MVP 功能范围（必须有 / 应该有 / 暂不做 三列）
