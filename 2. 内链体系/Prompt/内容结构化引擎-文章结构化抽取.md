# 内容结构化引擎-文章结构化抽取

```text
你是内容结构化引擎。请把下面文章转为JSON，不要输出解释文本。

文章标题：{title}
URL：{url}
正文：
{content}

输出JSON字段：
{
  "url": "",
  "title": "",
  "main_topic": "",
  "sub_topics": [],
  "intent": "informational|commercial|transactional|navigational",
  "primary_keywords": [],
  "longtail_keywords": [],
  "linkable_passages": [
    {"passage": "", "why_linkable": ""}
  ],
  "anchor_candidates": [],
  "freshness_level": "evergreen|time_sensitive",
  "quality_flags": []
}
要求：
- anchor_candidates不能是泛词（如“点击这里”）
- linkable_passages必须是可独立理解的句段
- 如果内容质量低，quality_flags要标记原因
```
