# 链接规划器-候选目标页排序

```text
任务：为源文章选择最合适的内部链接目标页。

源文章：
- title: {source_title}
- url: {source_url}
- main_topic: {source_topic}
- key_passages: {source_passages}

候选目标页列表（JSON）：
{candidate_pages_json}

请输出按优先级排序的前{K}个目标页，字段：
[
  {
    "target_url": "",
    "relevance_score": 0-100,
    "reason": "",
    "recommended_anchor_texts": [],
    "placement_hint": "intro|body|conclusion"
  }
]

约束：
- 禁止推荐与源URL相同页面
- 禁止主题冲突或关键词蚕食严重的页面组合
- 至少1个推荐给支柱页（如果语义相关）
- reason必须具体到语义点，不要泛泛而谈
```
