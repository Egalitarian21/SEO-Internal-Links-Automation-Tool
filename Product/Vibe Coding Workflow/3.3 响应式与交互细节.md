# Phase 2-3：响应式与交互细节

## 任务描述
为已完成的静态页面添加响应式布局支持和用户交互动效。
移动端需要折叠侧边栏，数据展示需要适配小屏幕。
使用 Framer Motion 添加微动画提升用户体验，但要保持克制，避免过度动效。

---

## Prompt

为 Dashboard 页面添加响应式支持和交互细节：

响应式要求：
- 移动端（< 768px）：侧边栏隐藏，顶部显示 hamburger 按钮，
  点击后侧边栏从左侧滑入（overlay 模式）
- 平板端（768px - 1024px）：侧边栏折叠为图标模式（宽度 64px）
- 桌面端（> 1024px）：侧边栏完整展示

交互动效（使用 framer-motion）：
- 页面初始加载：StatCard 依次从下向上淡入（stagger 0.1s）
- StatCard hover：轻微上移 + 阴影加深（translateY -2px）
- 侧边栏展开/折叠：平滑宽度过渡动画
- 表格行 hover：背景色过渡
- 表格行点击：跳转到 /projects/[id]，添加点击波纹效果

其他细节：
- 数字统计卡片的数值使用计数动画（从 0 增长到目标值）
- 图表加载时显示骨架屏（Skeleton）
- 添加键盘导航支持（Tab 键在菜单项间切换）
